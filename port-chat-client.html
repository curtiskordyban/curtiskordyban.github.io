Chat Room Client

  • View full project details on GitHub

This application is a simple form that simulates the functionality of a chat room. The user sends and receives messages by inputing the desired IP address, and has the option to include a username with the message.

Experience Gained

  • Used Udpclient sockets to exchange data
  • Contains asynchronous methods
  • Used regular expressions to parse out embedded username in message
  • Practiced various aesthetics such as text box placeholders
