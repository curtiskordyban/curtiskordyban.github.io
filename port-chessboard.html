Electronic Chessboard

With the current Covid-19 pandemic, this group project challenged me and my co-developer Christofer Tavano. We wanted to create a prototype electronic chessboard that would enable players to engage in a game of chess remotely. In addition, this chessboard has the capability to act as a great tool for beginners to educate them on the mechanics of each chess piece.

The idea of this project came to me one night while I was playing chess on a mobile application. As I was getting punished by an experienced player, I concluded that playing on a mobile app does not compare to the same nostalgic feeling of playing on a physical chessboard. I realized that people no longer sit down together and play chess with one another. We hope to resolve this issue by creating a chessboard that someone could have on display in their office or living room. A player could complete their turn during a morning coffee, go about their day and return once available again. We believe this is a unique way to maintain relationships and would be a product I would personally purchase.

Given the current circumstances of Covid-19, unfortunately we had to put this project on pause. After society returns to a more stable and safer environment, we hope to return to this project. We would like to create a second iteration of our concept and further improve our design.

Software Components

  • Used Ubuntu Server 20.04.01 LTS operating system on Raspberry Pi 3
  • Chessboard logic written in Python
  • Created a database using SQL to store and transmit game data
  • Set up a webservice using PHP to access database information
  • Created chessboard webpage using HTLM, CSS, and JavaScript
  • Used Ajax call requests to our API to retrieve game information for our webpage
